# Food Cart

A simple food ordering system built with Python that allows customers to browse a menu, select items, and calculate their total bill.

## Features

- Display menu with food items and prices
- Add items to cart
- Calculate total bill with itemized list
- Simple and intuitive command-line interface


## How to Run

```bash
python food_cart.py
```

Follow the on-screen prompts to select items and add them to your cart. Enter the item number to add it, or `0` to finish and view your bill.

## Code Structure

- `food_prices` dictionary - Stores menu items and their prices
- `items` list - Tracks selected items in the cart
- `total` variable - Keeps running total of the bill

The program uses a simple loop to display the menu, accept user input, and calculate the final total.

## Requirements

- Python 3.x

No external dependencies required - uses only Python standard library.
