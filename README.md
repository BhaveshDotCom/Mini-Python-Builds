# Mini Python Builds 

This repository contains small Python builds created while learning.
Each project focuses on implementing concepts through practical code.

New builds are added regularly.

## 📋 Builds

| No. | Build Name              | Description                               |
| --- | ----------------------- | ----------------------------------------- |
 | 1   | Cipher Tool | Simple tool for encoding and decoding messages |
 | 2   | Email Slicer | Extracts username and domain from email addresses |
 | 3   | Timer | Countdown timer with DD:HH:MM:SS format |
 | 4   | Food Cart | Food ordering system with menu and cart functionality |
 | 5   | QR Code Generator | Generates QR codes from text or URLs |
